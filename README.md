# cars_basicTF

The project contains a solution for folowing task:
- Go to www.cars.com.
- Fill search form with:
  - make "McLaren"
  - model "650S"
  - radius "All Miles"
  - zip "60026"
- Hit Search button.
- Check numbers of results.
- Change pagination to "10 per page".
- Assign that numbers of results are different. 
  
Techs: 
- java
- Selenium Web-driver
- page objects
